Voor ons eindproject hebben wij, Wout Vandevelde en Noah Vermaerke, een unieke robot ontwikkeld met mecanum wielen. Dankzij deze wielen kan de robot soepel in alle richtingen bewegen, inclusief zijwaarts. De besturing gebeurt via een zelfgemaakte afstandsbediening die communiceert met de robot via WiFi. Dit project combineert mechanische innovatie en draadloze technologie.


Globale werking van het project

Het project omvat een robot met een zelfgemaakte afstandsbediening.

Het blokschema is opgebouwd uit:

  • Het robotframe met de raspberry pi (mini-computer) en een powerbank.
  • De router voor het draadloos netwerk.
  • Het dashboard (voor de bediening van de robot op de laptop).
  • De afstandsbediening met de joysticks, batterij en een schakelaar.
