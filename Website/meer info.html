De raspberry Pi (mini-computer)

De Raspberry Pi is een server waarop er software (=Node-Red) actief is zodat een smartphone, tablet of PC het dashboard kan zien. Op de server draait ook een Mosquitto broker die nodig is voor de communicatie tussen de robot en NodeRed.

Node-Red


Node-RED is de software die draait op de Raspberry Pi, de laptop en ook gebruikt wordt voor het maken van het dashboard.

Node-RED is een visuele programmeertool voor het verbinden en automatiseren van hardware, API's en online diensten. Het maakt gebruik van een flow-gebaseerde interface, ondersteunt protocollen zoals MQTT, en is ideaal voor IoT-toepassingen.

Via een Mosquitto broker verstuurt Node-RED de gegevens van het dashboard draadloos naar de Raspberry Pi.

Het is open source, uitbreidbaar en compatibel met verschillende platforms.

Mosquitto broker

De Mosquitto broker is software die wordt gebruikt om draadloos te communiceren tussen Node-RED, de Raspberry Pi en de microcontroller.

Een MQTT-broker, zoals Mosquitto, dient als een centrale server binnen het MQTT-protocol, dat vaak wordt ingezet in IoT (Internet of Things). De broker zorgt ervoor dat berichten van publishers bij de juiste subscribers terechtkomen.
