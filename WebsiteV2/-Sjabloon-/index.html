Robot Met Mecanum Wielen


Voor ons eindproject hebben wij, Wout Vandevelde en Noah Vermaerke, een unieke robot ontwikkeld met mecanum wielen. Dankzij deze wielen kan de robot soepel in alle richtingen bewegen, inclusief zijwaarts. De besturing gebeurt via een zelfgemaakte afstandsbediening die communiceert met de robot via WiFi. Dit project combineert mechanische innovatie en draadloze technologie.

Globale uitleg van het project

Globale werking

Het project omvat een robot met een zelfgemaakte afstandsbediening.

Het blokschema is opgebouwd uit:

  • Het robotframe met de Raspberry Pi (mini-computer), een powerbank
  • De router voor het draadloos netwerk
  • Het dashboard (voor de bediening van de robot op de laptop)
  • De Afstandsbediening met joysticks, potentiometer, batterij en een schakelaar

    • Het Robotframe:

    Het robotframe bestaat uit:

  • Het frame
  • 4 DC-motoren (elektrische kleine motoren die de wielen laten draaien)
  • 4 mecanum wheels (wielen die alle richtingen op kunnen)
  • Een powerbank en een Raspberry Pi (een minicomputer)

    • De Raspberry Pi krijgt signalen door van de afstandsbediening via het draadloos netwerk (Wi-Fi) en stuurt zo de DC-motoren aan om de mecanum wielen te laten draaien en zo de robot de juiste richting te laten rijden.

    De powerbank is makkelijk op te laden met een USB-kabeltje, en als je de robot uit wil schakelen, trek je gewoon de kabel uit de powerbank.

    De Controller (Afstandsbediening):

    De controller bestaat uit een micro-controller (de ESP32 Feather), 2 joysticks, een potentiometer, een batterij en een aan- en uit schakelaar.

    Met de 1ste joystick stuur je de robot naar voor en achter, links en rechts. Met de 2de joystick bepaal je hoe de robot rond zijn as draait. De micro-controller ontvangt wat de positie van de joysticks zijn, verwerkt deze en stuurt deze door via WiFi naar de Raspberry Pi op de robot.

    De batterij is aangesloten op de grote connector van de micro-controller en voorziet deze van spanning. Als de batterij leeg is, dan moet de micro-controller aangesloten worden op de USB-poort waardoor de batterij kan opladen. Als de micro-controller niet gebruikt wordt dan kan de schakelaar gebruikt worden om zo de micro-controller uit te zetten, zodat de energie van de batterij niet wordt verspild.

    De robot kan niet alleen aangestuurd worden met de controller, maar ook met de knoppen op het dashboard. Dit wordt verder uitgelegd.

    Besturingen met de controller

    Joystick 1 (links):

    De robot gaat dezelfde richting uit als de joystick (zonder te draaien).

    Joystick 2 (rechts):

    De robot draait links om zijn as als je de joystick naar links duwt, hetzelfde om naar rechts te draaien.

    Het dashboard:

    Het dashboard is een soort website/app waarop men bepaalde aansturingen kan doen. Je kan op het dashboard de robot vooruit, achteruit, links en rechts laten rijden, en ook rond zijn as laten draaien. Met de controller kan je dus meer precieze richtingen op gaan met de robot.

    Door op de pijltjes te klikken - of op het draai icoontje te klikken - en de slider van de maximum snelheid te verhogen kan je de robot laten rijden. Als je de robot wilt laten stoppen druk je op het vierkante knopje in het midden, of zet je de slider op 0.

    Als je het dashboard wilt gebruiken in plaats van de controller, zet je de controller gewoon uit vie da schakelaar op de achterkant.

    De router:

    De router zorgt voor een draadloos netwerk dat gebruikt wordt voor de communicatie van de robot en de tablet, smartphone of PC waarop het dashboard, de Raspberry Pi en de microcontroller draait.

    De Raspberry Pi (mini-computer):

    De Raspberry Pi is een server waarop er software (=Node-Red) actief is zodat een smartphone, tablet of PC het dashboard kan zien. Op de Raspberry Pi draait ook een Mosquitto broker die nodig is voor de communicatie tussen de robot en NodeRed.

    Node-red:

    Node-red is de software die op de Rapsberry Pi draait. Het dashboard is er ook mee gemaakt. En kan je gebruiken op elk toestel verbonden met hetzelfde WiFi-netwerk.

    Het verstuurt ook de gegevens van op het dashboard via een mosquitto broker naar de Raspberry Pi via het draadloos netwerk.

    Mosquitto broker:

    Mosquitto broker is een software waar Node-red, de Raspberry Pi en de microcontroller gebruik van maakt om te communiceren met MQTT-berichten via het draadloos netwerk.
